FreeCell DS

About

FreeCell DS is a FreeCell game for the Nintendo DS. Cards are moved around using the stylus. Deal a new deal by pressing the start button.

The game is still in development, but it works well enough to play.

The game does support the same deals as in Windows FreeCell, but there is currently no way to select a specific game.
